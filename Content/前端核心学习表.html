前端学习计划表

周数 主题 核心任务 时间分配 产出物
第1周 HTML + CSS - 掌握常用标签（div, span, a, form, input 等）
- CSS 盒模型、选择器优先级
- Flex 布局、定位（relative/absolute/fixed）
- 实现：两栏/三栏布局、导航栏、水平垂直居中		 每天 2–3 小时 3个静态页面（如个人主页、博客首页、商品列表）
第2–3周 JavaScript（重点！） - 变量、函数、循环、对象、数组方法
- ES6+：解构、箭头函数、Promise、async/await
- DOM 操作 + 事件绑定
- 手写：call/bind、debounce/throttle、深拷贝
- 练习：轮播图、Tab 切换、TodoList		 每天 3–4 小时 2个交互式小项目 + 手写 API 笔记
第4周前3天 工程化入门 - 安装 Node（用 nvm）、npm 基础命令
- 了解 ESM/CJS 模块化
- 用 Vite 创建 Vue3 或 React 项目
- 学习 TS 基础类型（string/number/interface/type）		 每天 2 小时 能运行一个带 TS 的框架项目
第4周后4天 前端框架（二选一） Vue3 路线：
- ref/reactive、computed/watch
- 组件通信（props/emits、provide/inject）
- Vue Router + Pinia
React 路线：
- JSX、useState/useEffect/useCallback
- Context + 自定义 Hook
- React Router + Zustand（轻量）		 每天 3 小时 一个带路由和状态管理的 Todo 应用
第5–6周 实战项目 + 应试准备 - 开发 2个完整项目（建议：1个业务型 + 1个工具型）
- 集成：登录鉴权、API 请求、错误处理
- 添加：Eslint/Prettier、部署到 Vercel
- 同步整理 八股文笔记（HTML/CSS/JS/网络）
- 刷 LeetCode Hot 100（至少 Easy 全部）		 每天 4 小时 2个 GitHub 项目 + 1份简历 + 八股知识库

💡 关键提醒

  • 第3周结束前必须能独立写 JS 逻辑，否则框架学得吃力。
  • 项目不必"高大上"，但要 能讲清楚技术选型、难点、优化点。
  • 每天留 30 分钟复盘：今天学了什么？哪里卡住了？明天怎么解决？

八股文重点

  • JS：闭包、this、原型链、事件循环、Promise 原理
  • 浏览器：重绘/回流、缓存机制、跨域
  • 网络：HTTP 状态码、TCP 三次握手、HTTPS

推荐资源

  • 前端面试高频八股文（掘金）
  • LeetCode Hot 100（刷 Easy + Medium）

第一阶段：夯实基础（第 1–2 周）

目标：掌握 Java、SQL、HTML/CSS/JS 核心语法，能写简单程序。

技术点 学习内容 推荐资源
Java 基础 变量、循环、方法、类与对象、集合、异常处理 廖雪峰 Java 教程 / B站"韩顺平 Java 零基础"
SQL 与数据库 SELECT/INSERT/UPDATE/DELETE、JOIN、索引概念；安装 MySQL 并练习 《MySQL 必知必会》/ 菜鸟教程 SQL
前端三件套 HTML 结构、CSS 布局（Flex/Grid）、JavaScript 基础语法（变量、函数、DOM 操作） MDN Web Docs / freeCodeCamp 中文

✅ 实践任务

  • 用 Java 写一个学生管理系统（控制台版）；
  • 在 MySQL 中建表并完成增删改查；
  • 用 HTML+CSS+JS 做一个静态个人简介页（响应式布局）。

第二阶段：后端框架入门（第 3–5 周）

目标：掌握 Spring Boot + MyBatis 开发 RESTful API。

技术点 学习内容
Spring Boot 项目创建、Controller、Service、自动配置、RESTful 接口设计
MyBatis / MyBatis-Plus ORM 映射、Mapper 接口、动态 SQL
数据库进阶 事务、连接池、多表关联查询
开发工具 IDEA 创建 Spring Boot 项目、调试、热部署

✅ 实践任务

  • 开发一个"图书管理系统"后端：
    • 提供 /api/books 的 GET/POST/PUT/DELETE 接口；
    • 使用 MyBatis 操作 MySQL；
    • 用 Postman 测试接口。

📌 提示：可使用 Spring Initializr 快速生成项目骨架。

第三阶段：前端框架入门（第 6–8 周）

目标：用 Vue 实现前端页面，调用后端 API。

技术点 学习内容
Vue 3 基础 响应式数据、指令（v-if/v-for）、组件化、生命周期
Vue Router & Axios 页面路由跳转、发送 HTTP 请求
UI 组件库 Element Plus 或 Ant Design Vue（选一个）
前后端联调 跨域问题（CORS）、接口对接

✅ 实践任务

  • 为"图书管理系统"开发前端页面：
    • 列表页（展示所有书）
    • 新增/编辑弹窗
    • 使用 Axios 调用后端接口
    • 用 Element Plus 美化界面

📌 工具：VS Code + Vite 创建 Vue 项目（比 Vue CLI 更现代）。

第四阶段：整合与提升（第 9–12 周）

目标：完成一个完整小项目，模拟真实开发流程。

✅ 综合项目建议

  • 校园二手交易平台（用户发布商品、浏览、搜索）
  • 个人博客系统（文章管理、评论）

涵盖技能

  • 前后端分离架构
  • 用户登录（可先用 Session，后续学 JWT）
  • 分页、搜索、图片上传（基础实现）
  • 使用 Git 管理代码（GitHub/Gitee）

📌 同时补充软技能：

  • 学习 Git 基本命令（clone, add, commit, push, branch）
  • 阅读他人开源项目（如 GitHub 上的 springboot-vue-demo）
  • 尝试写技术笔记或项目 README，锻炼表达能力
