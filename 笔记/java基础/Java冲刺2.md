# Java冲刺2

###### 3小时雷打不动，断一天要补两天

- **1小时**：看视频/看书学新知识
- **1.5小时**：动手敲代码（必须！）
- **0.5小时**：复习/总结/记笔记

**周末可适当延长**，补本周落下的内容或做小项目。

------

## 📚 阶段一：Java 核心基础（5周）

**目标**：掌握变量、流程控制、数组、面向对象、集合、异常、IO流（简单了解）

### 每周安排

| 周    | 学习内容                                                     | 每日练习任务                              |
| :---- | :----------------------------------------------------------- | :---------------------------------------- |
| 第1周 | Java环境搭建、变量、数据类型、运算符、Scanner输入            | 每天写3-5个简单程序（计算器、判断奇偶等） |
| 第2周 | 条件语句（if/else, switch）、循环（for, while）、break/continue | 打印各种图形、猜数字游戏                  |
| 第3周 | 数组、方法（定义、调用、重载）                               | 数组排序、求最大值、方法封装              |
| 第4周 | 面向对象：类与对象、构造方法、封装、this                     | 设计学生类、图书类，创建对象              |
| 第5周 | 继承、多态、接口、常用类（String, ArrayList）                | 实现简单继承体系、使用集合存储数据        |

**资源**：B站“黑马Java基础”（每天看2-3集，跟着敲）
**注意**：此阶段一定要多敲，每行代码都手打，不要复制。

------

## 🗃️ 阶段二：MySQL + JDBC（2周）

**目标**：会写SQL增删改查，能用JDBC连接Java操作数据库

### 安排

| 周    | 学习内容                                                     | 每日练习                                 |
| :---- | :----------------------------------------------------------- | :--------------------------------------- |
| 第6周 | MySQL安装、SQL基础（SELECT, INSERT, UPDATE, DELETE）、WHERE、ORDER BY | 建表（学生表、成绩表），练习增删改查     |
| 第7周 | 多表查询（内连接、外连接）、JDBC原理、PreparedStatement      | 用JDBC写一个简易学生管理系统（控制台版） |

**资源**：B站“尚硅谷MySQL基础” + “JDBC快速入门”
**重点**：JDBC是后面框架的基础，一定要理解连接、执行SQL、关闭资源的过程。

## 🌱 阶段三：Spring Boot 后端开发（6周）

**目标**：能独立搭建Spring Boot项目，写REST接口，整合MyBatis操作数据库

### 安排

| 周     | 学习内容                                                     | 每日练习                                           |
| :----- | :----------------------------------------------------------- | :------------------------------------------------- |
| 第8周  | Maven/Gradle构建工具、Spring Boot入门（创建项目、启动类、配置文件） | 创建第一个Spring Boot项目，写一个Hello接口         |
| 第9周  | Controller、Request参数接收、返回JSON、Postman测试           | 写一个用户Controller，实现增删改查接口（内存存储） |
| 第10周 | 整合MyBatis（或JPA）、Mapper接口、XML或注解SQL               | 将之前的接口改为数据库版，实现真实CRUD             |
| 第11周 | 依赖注入（@Autowired）、Service层拆分、事务管理              | 重构代码，分层清晰，加入事务                       |
| 第12周 | 统一异常处理、日志、配置文件多环境                           | 给项目添加全局异常和日志                           |
| 第13周 | 项目实战：开发一个简单的“博客系统”后端（用户、文章、评论）   | 完成所有接口，用Postman测试                        |

**资源**：B站“尚硅谷SpringBoot2”（只看基础篇，约50集，每天2-3集）
**练习**：每个小知识点都要自己实现一遍，最后完成博客后端。

**目标**：会用Git进行版本控制，能看懂Vue前端代码，会调用后端接口

## 🎯 面试准备（最后1周+穿插进行）

- **Java基础**：集合、多线程、JVM（常见面试题）
- **Spring Boot**：自动配置、常用注解、事务
- **数据库**：索引、事务隔离级别、SQL优化
- **项目**：准备一个你最熟悉的项目，能讲清楚技术点、难点、AI怎么实现的
- **算法**：LeetCode简单题（可选，但岗位一般不问）

面试前一周，每天模拟面试，自己讲项目。

### 1. Java基础（必问）

| 知识点 | 常见问题                                                  |
| :----- | :-------------------------------------------------------- |
| 集合   | ArrayList和LinkedList区别、HashMap原理、ConcurrentHashMap |
| 多线程 | 线程创建方式、synchronized和Lock区别、线程池参数          |
| JVM    | 内存区域、垃圾回收算法、类加载机制                        |
| 异常   | 运行时异常和检查异常区别                                  |
| 反射   | 什么是反射、应用场景                                      |

### 2. Spring Boot/Spring（必问）

| 知识点   | 常见问题                             |
| :------- | :----------------------------------- |
| IOC      | 什么是IOC、Bean的生命周期            |
| AOP      | 原理、应用场景（日志、事务）         |
| 自动配置 | @SpringBootApplication原理、条件注解 |
| 事务     | @Transactional失效场景、传播机制     |

### 3. 数据库（必问）

| 知识点  | 常见问题                         |
| :------ | :------------------------------- |
| 索引    | 索引类型、B+树原理、最左前缀原则 |
| 事务    | ACID、隔离级别、MVCC             |
| SQL优化 | 慢查询排查、explain分析          |
| 锁      | 行锁、表锁、乐观锁、悲观锁       |

### 4. 计算机网络（常问）

- OSI七层模型、TCP/IP四层模型
- TCP三次握手、四次挥手
- HTTP和HTTPS区别
- GET和POST区别

### 5. 设计模式（常问）

- 单例模式（手写）
- 工厂模式
- 代理模式（静态代理、动态代理）
